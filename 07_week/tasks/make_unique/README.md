# Создать умный указатель

Необходимо реализовать шаблонную функцию `MakeUnique`, которая принимает параметры
конструктора объекта и создает умный указатель `std::unique_ptr` на созданный 
объект в динамической памяти.

Поддержка работы с массивами не требуется.

## Примечание

- **Запрещено** использовать стандартную функцию `make_unique`
- Передача аргументов должна быть _perfect forwarding_
- Функция должна принимать произвольное число аргументов
